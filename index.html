辗转相除法

发表于 2021-11-10 更新于 2021-11-15 分类于 18CD02CPlusPlus

1. 辗转相除法（欧几里得算法） 求最大公约数

举例：求 a=72,b=39 的最大公约数（Greatest Common Divisor, 简称Gcd）
a % b = rem
72 % 39 = 33
39 % 33 = 6
33 % 6 = 3
6 % 3 = 0
3 % 0
